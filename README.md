## how to test ?

add remote or download source then :

**TLDR**

```bash
flutter pub get
```
```bash
flutter pub get
```
```bash
flutter pub run
```
used api :

* website: `CoinCap.io`
* api url: `https://rest.coincap.io/v3/assets?apiKey=${apiKey}`

used package in Crypto :

* Dio: `dio: ^5.8.0+1`
* SpinKit: `flutter_spinkit: ^5.2.1`
* Luncher Icon: `flutter_launcher_icons: ^0.14.3`
